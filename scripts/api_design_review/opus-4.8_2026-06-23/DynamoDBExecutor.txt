1. OVERALL ASSESSMENT
The v1 DynamoDB executor is a large, coherent thin wrapper over the AWS SDK v1 AmazonDynamoDB API. Like
HBaseExecutor (and unlike the mongo/cassandra get=Optional discipline), it intentionally returns SDK
*Result types and bare nullable T/Map for single-item reads — an SDK-mirror style that is internally
consistent and should NOT be re-flagged as an Optional violation. Findings are cross-twin (v1 vs v2)
naming/parity items.

  AttributeAction]) | DynamoDBExecutor.java:906,949,982 | High | Drift vs the v2 twin's attrValueOf /
