1. OVERALL ASSESSMENT
ColumnFamily is a clean, correctly-designed mapping annotation. The single value() element with an empty
default enables the `@ColumnFamily("cf")` shorthand, and HBaseExecutor treats "" as "unset"
(HBaseExecutor.java:~539-540). Together with @Column it forms a tidy family/qualifier split. Only minor
Javadoc-accuracy items.

2. NAMING ISSUES
None — value() (not name()) is the correct element name for the single-value shorthand.
