1. OVERALL ASSESSMENT
Neo4jExecutor faithfully mirrors the Neo4j OGM Session (load/loadAll/save/delete/count) plus a Cypher
query surface. Cypher parameter-passing is clean ((String cypher, Map<String,?> params) with $name
binding; no unsafe Object[] varargs). Session/pool ownership is correct and well documented. No
correctness bugs. The OGM-native verbs (load/loadAll) are an intentional faithful mirror and should stay;
the only structurally significant item is a query() return type that collides with the library's
query/stream convention. Remaining items are optional additive family wrappers + a dead comment.

 naming-convention commit did NOT rename query()->Stream to stream() (no stream() method exists). No
 get()/findFirst()/count(Class)/saveAll()/bulk-delete added; dead queryDto comment still present. Only
 stale line numbers corrected below.]
